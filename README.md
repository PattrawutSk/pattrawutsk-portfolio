## 🚀 Deployment Flow

This project is automatically deployed using Cloudflare Pages:

1. Update code locally
2. Upload / push to GitHub repository
3. Cloudflare detects changes automatically
4. Auto build & deploy within ~1 minute

No manual deployment needed.
